Sports Playground Booking System this System shows the available sports facilities/arena’s to the User and also provides the function of selecting the required timing slot. Through this system the users can easily and securely authenticate themselves to the booking websites through log-in page. Here, the admins can add/delete the sports facilities and can also add/update the pricing of it and this system provides the admin with the feature of confirmation/cancellation of the user’s booking. Admin can also view the total number of user’s registered with the booking system and their booking details also.﻿
